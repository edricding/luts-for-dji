# LUTs for DJI

本仓库用于整理 DJI 相关 LUT 文件，当前分为两类：

- `offical/`：官方技术转换 LUT
- `creative/`：创意风格 LUT（已统一英文命名，前缀为 `lut-`）

## 目录概览

- `offical/`：1 个 LUT
- `creative/`：17 个 LUT

## Offical LUT 列表

> 目录名为 `offical`（与仓库当前目录保持一致）

| 文件名 | 说明 |
| --- | --- |
| `D-Log M to Rec.709 V1.cube` | DJI D-Log M 到 Rec.709 的官方转换 LUT，适合做基础色彩还原/技术转换，适用于DJI所有系列的D-log灰片还原。 |

## 命名规则

`creative/` 目录已统一使用以下命名规则：

- 前缀：`lut-`
- 风格：英文小写单词
- 分隔：`-`
- 扩展名：`.cube`

示例：`lut-kodak-5219-warm.cube`
